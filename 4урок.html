3 URL

<схема>://<логин>:<пароль>@<хост>:<хост>:<порт>/<путь>?<параметр>#<якорь>    http    ftp    file    chrome    mailto    skype
схема - - это якорь для открытия в соов приложении http - браузер file - открыть файл skype - открывает программу скайп и т.д.
логин и пароль - это соответственно логин и пароль, если их нет то они опускаются
хост - устройство подключенное к сети не важно какое (телефон, компьютер и тд) он описывается его IP адресом
ipv4 127.0.0.1
или
ipv6 338d:0242:0000:0000:0000:0000:1156:200F:35ce
четыре нуля подряд меняются на просто два двоеточия ::
вместо доменного имени может быть хост! то есть просто его ip
но как правило - всегда доменное имя
доменное имя через DNS приравнивается к определенному ip
* корневой домен
.com .org. net - бомены второго уровня
потом если стоит точка после второго уровня идет домен третьего уровня .ua .ru и тд
есть и четвертого уровня .kiev .kaharkov и тд

порт - это порт

путь - это путь к папке (файлу) от корня сервера слешь - / это корень папки
параметры - это все что находится после знака вопроса каждый параметр состоит из пары - ключ и значение
значения параметров как правило закодированы в unicode все эти занки процента + цифры - это закодированные в
юникод кирилические символы
ВАЖНО!
внутри ссылки не может быть записана ссылка в кирилице - только в unicode варианте.
# якорь - это якорь
Все это описание полного URL пути ))
